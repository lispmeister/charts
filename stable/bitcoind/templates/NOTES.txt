lseed can be accessed via port {{ .Values.lseed.service.lseedPort }}
on the following DNS name from within your cluster:
{{ template "bitcoind.fullname" . }}.{{ .Release.Namespace }}.svc.cluster.local

To connect to lseed:

1. Forward the port for the node:

  $ kubectl port-forward --namespace {{ .Release.Namespace }} $(kubectl get pods --namespace {{ .Release.Namespace }} -l "app={{ template "bitcoind.name" . }},release={{ .Release.Name }}" -o jsonpath="{ .items[0].metadata.name }") {{ .Values.lseed.service.lseedPort }}

2. Test connection with a DNS query:

$ nslookup node.lightning.directory 127.0.0.1:{{ .Values.lseed.service.lseedPort }}
